# Learning-APIs-Through-Postman
Project Overview
This repository contains API tests for the OpenCart REST API using Postman.
The goal of this project is to practice and demonstrate:
REST API testing fundamentals
Authentication handling
CRUD operation validation
Environment variable usage
Automated test scripting in Postman


Tools Used:
Postman
OpenCart (Local setup via XAMPP)
Newman (Postman CLI runner)
Git & GitHub

Test Coverage:
The following areas are being tested:
Authentication (Login & Token Handling)
Product APIs (Create, Read, Update, Delete)
Status Code Validation
Response Structure Validation
Edge Case & Negative Testing

Project Structure:
/collections → Postman Collection export
/environments → Postman Environment export
/docs → Test reports or screenshots (if applicable)
